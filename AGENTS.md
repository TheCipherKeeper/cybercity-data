# AGENTS.md — сервис cybercity-data

Репозиторий одного независимо поставляемого Python-сервиса. Общие полномочия,
рабочий цикл, архитектурный канон и поставка заданы закреплённой методологией:
`<methodology-repo>/AGENTS.md` и `<methodology-repo>/docs/`. Системный состав,
контракты и ADR находятся в хабе `TheCipherKeeper/cybercity`.

## Локальные правила

- Назначение сервиса, публичные файловые артефакты и события не меняются без
  явно поставленной человеком задачи.
- `organizations/` содержит исходную декларативную модель города; валидатор
  нельзя ослаблять ради прохождения некорректных данных.
- ADR и системные контракты изменяются только в хабе.
- `.env`, `uv.lock`, сгенерированный `build/` и секреты без явного разрешения
  не изменяются.
- Код, тесты, `docs/ARCHITECTURE.md` и применимые спеки обновляются согласованно.

## Команды

| Стек | Проверка | Тест | Сборка |
|---|---|---|---|
| Python | `uv run ruff format --check . && uv run ruff check . && uv run mypy --strict src/cybercity_data` | `uv run pytest -q` | `uv build` |

## Коммиты и язык

Conventional Commits; область — модуль либо `docs`/`deploy`. Документация и
поясняющий текст — русские; точные идентификаторы и краткое описание коммита
могут быть английскими. Несовместимость отмечается `BREAKING CHANGE:`.
