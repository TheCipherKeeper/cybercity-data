# ADR-0003: Явные сети и IP-адреса в v2.0

## Status

Superseded (см. [ADR-0016](0016-generated-networks-allocator.md))

## Context

В v2.0 сети и IP-адреса декларировались явно в YAML-модели (см. CHANGELOG
0.3.0: «Явные сети и IP-адреса в v2.0»).

## Decision

(Историческое, v2.0.) Сети и IP-адреса задаются явно в декларативной модели.

## Consequences

- Вела к адресной бухгалтерии внутри декларативного слоя.
- В v3.0 заменена генерацией адресации аллокатором: модель описывает только
  топологию (см. ADR-0016 и ARCHITECTURE.md «Модель данных (v3.0)»).
