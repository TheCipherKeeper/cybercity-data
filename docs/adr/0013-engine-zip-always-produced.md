# ADR-0013: `engine.zip` генерируется всегда, даже без ассетов

## Status

Accepted

## Context

Контракт **data → engine** не должен зависеть от наличия ассетов сервисов:
engine всегда должен получать пакет.

## Decision

`engine.zip` генерируется при каждой сборке, даже когда ассеты сервисов
отсутствуют.

## Consequences

- Контракт стабилен; engine всегда получает пакет.
- Добавлено в Unreleased (CHANGELOG: «`engine.zip` теперь генерируется при
  каждой сборке»).
